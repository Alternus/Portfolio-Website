######################
To run the project locally install php and open a terminal/console in the folder and run:

php -S 127.0.0.1:8080

Website will be avalible locally on https://127.0.0.1:8080

#########################

! Note ! database fetches will be unusable locally unless you un-comment line 3 and comment line 2 on the following php documents as these files are configured for the live site

api/admin.php
api/messages.php

#########################

To build the project after modifying ReactJS files within src/ install NodeJS and open a terminal/console in the folder and run:

npm install
npm run build
